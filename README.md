# Advanced To-Do List Application

The Advanced To-Do List Application is a modern task management web application built using **HTML, CSS, and JavaScript**. It helps users organize daily activities through an intuitive and responsive interface while demonstrating core front-end development concepts.

The application allows users to create, edit, delete, and mark tasks as completed. Each task can include a priority level, category, and due date, making it easier to manage personal, academic, or work-related activities. Users can quickly search for tasks, filter them by status, and monitor progress through a dashboard that displays total, completed, and pending tasks along with a visual progress bar.

To improve user experience, the application includes Dark/Light Mode, Local Storage for persistent data, drag-and-drop task reordering, JSON import/export functionality, toast notifications, and a responsive design that works across desktop and mobile devices.

This project demonstrates practical JavaScript concepts such as DOM manipulation, event handling, Local Storage, dynamic rendering, array methods, modular programming, and responsive UI development. It serves as a strong portfolio project for showcasing front-end development skills and modern web application design.
